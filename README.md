# COVID-19 Data Warehouse & ETL (MySQL + Python)

## Overview

An end-to-end **ETL and Data Warehouse project** that extracts raw COVID-19 data, transforms it into a **star schema**, and loads it into **MySQL** for analytics and visualization.

## Tech Stack

* Python (pandas, sqlalchemy)
* MySQL (Workbench)
* SQL
* Matplotlib / Seaborn
* VS Code

## Data Model

* **Dimensions:** `dim_country`, `dim_date`
* **Fact:** `fact_covid`

## ETL Flow

```
CSV Data ‚Üí Python (ETL) ‚Üí MySQL Data Warehouse ‚Üí Analytics & Visualization
```

## Features

* Star schema design
* Automated ETL pipeline
* SQL-based analytics
* Python visualizations

## How to Run

1. Run SQL schema from `/sql`
2. Execute ETL scripts from `/src`
3. Analyze data in `/notebooks`
